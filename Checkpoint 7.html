Checkpoint 7

JavaScript es un lenguaje de programación desarrollado en el año 1995 por la empresa Netscape con el objetivo de aportar mayor dinamismo a las páginas web.

JavaScript es el único lenguaje de programación que puede ser interpretado por los navegadores web modernos, es por eso que se considera la tercera capa del desarrollo de tecnologías web estándar:

  • Primera capa - HTML: lenguaje de marcado utilizado para estructurar y dar significado al contenido de una web.
  • Segunda capa - CSS: lenguaje de reglas de estilo para aplicar estilos al contenido HTML.
  • Tercera capa - JavaScript: lenguaje de programación usado para añadir dinamismo y funciones que mejoran la experiencia del usuario.

Tipos de datos

JavaScript es considerado, comprado con otros, un lenguaje de programación débilmente tipado y con tipado dinámico, es decir, que las variables no están directamente asociadas a un tipo de dato y, además, se les puede asignar y reasignar un tipo de dato distinto en cualquier momento.

Estos son algunos de los tipos de datos que existen en JavaScript:

  • Booleanos (boolean):

    Representa un valor lógico y puede tener dos valores, true o false.

    Un ejemplo de su uso sería comprobar si un usuario de una plataforma de streaming ha pagado la suscripción o no.

    var pagado = true;
var noPagado = false;
  • Nulo (null):

    El único valor que puede tomar es null y representa la ausencia de cualquier valor.

    Si, por ejemplo, en un formulario un usuario deja sin completar un campo, la ausencia de contenido en este campo se representaría con null.

    var nombre = 'Alazne';
var correo = 'checkpoint@checkpoint.com';
var web = null;
  • Indefinido (undefined):

    Es una varible la cual se declara pero no se le asigna ningún valor.

    Este tipo de dato es utilizado por los programadores para localizar errores y depurar el código.

    var indefinido;
  • Número (number):

    Representa valores numéricos enteros y decimales.

    var entero = 1;
var decimal = 5.25;

    Con este tipo de dato se pueden realizar diversos cálculos matemáticos en JavaScript; algunos de los operadores matemáticos básicos que se pueden utilizar son estos:

    //Suma
console.log(2 + 2); //4
//Resta
console.log(5 - 3); //2
//División
console.log(9 / 3); //3
//Multiplicación
console.log(4 * 3); //12
//Potencia
console.log(2 ** 3); //8
//Resto o residuo
console.log(11 % 2); //1
  • Cadena de caracteres (string):

    Se utiliza para representar datos textuales. Cada uno de los caracteres es un elemento de la cadena, y la posición del elemento se define a través del índice, el primer caracter está en el índice 0, el segundo en el 1, y así sucesivamente.

    var cadena = 'Lorem ipsum dolor sit amet, consectetur adipiscing';

    Al igual que con los números, con las cadenas de valores también se pueden hacer distintas acciones o tareas, estas son las principales:

    //Convertir a mayúsculas
console.log(cadena.toUpperCase()); //'LOREM IPSUM DOLOR SIT AMET, CONSECTETUR ADIPISCING'

//Convertir a minúsculas
console.log(cadena.toLowerCase()); //'lorem ipsum dolor sit amet, consectetur adipiscing'

//Concatenar
console.log(cadena.concat(' elit')); //'Lorem ipsum dolor sit amet, consectetur adipiscing elit'

//Reemplazar
console.log(cadena.replace('ipsum', 'lorem')); //'Lorem lorem dolor sit amet, consectetur adipiscing'

//Localizar caracter según su índice
console.log(cadena.charAt(4)); //'m'

//Localizar índice según caracter o string
console.log(cadena.indexOf('amet')); //22

    A pesar de ser una propiedad y no una función, el atributo length también es muy utilizado para conocer el número de caracteres que tiene una cadena.

    console.log(cadena.length); //50
  • Objeto (object):

    Son coleciones de propiedades identificadas mediante valores clave.

    var datos = {
	nombre: 'Alazne',
	apellido: 'Portal'
	}

Se puede comprobar el tipo de dato de una variable a través del operador typeof.

console.log(typeof pagado); //boolean
console.log(typeof web); //null
console.log(typeof indefinido); //undefined
console.log(typeof entero); //number
console.log(typeof cadena); //string
console.log(typeof datos); //object

Además, JavaScript permite convertir los datos de un tipo a otro en función de para qué se necesite la variable:

  • Conversión a string: Convierte un valor a cadena de caracteres.
    console.log(noPagado.toString()); //'false'
console.log(entero.toString()); //'10'
  • Conversión a número: Convierte un valor a número.
    console.log(Number(noPagado)); //0
  • Conversión a booleano: Convierte un valor a booleano.
    console.log(Boolean(entero)); //true

Condicionales

Los condicionales son declaraciones que permiten a las aplicaciones ser más dinámicas gracias a que consienten al código realizar decisiones y llevar a cabo distintas acciones en función de si se cumplen o no las condiciones declaradas.

La sintáxis básica de un condicional es la siguiente:

if (condición) {
	acción condición verdadera
} else {
	acción condición falsa
}

Para establecer la condición, existen varios operadores de comparación:

a == b //igualdad
a === b //igualdad exacta
a !== b //desigualdad
a > b //mayor que
a >= b //mayor o igual que
a < b //menor que
a <= b //menor o igual que

Un ejemplo sencillo de un condicional sería este:

var num = 20;

if (num % 2 == 0) {
	console.log('Número par');
} else {
	console.log('Número impar');
}
//'Número par'

JavaScript también permite crear condicionales compuestos, los cuales tienen más de una condición para una misma acción; para crear estas condiciones compuestas se utilizan los operadores and (&&) y or (||).

var num = 20;

if (num % 2 == 0 && num !== 2) {
	console.log('Número par');
  console.log('No es número primo');
} else {
	console.log('Número impar');
}
//'Número par'
//'No es número primo'

Los condicionales se pueden expresar de forma abreviada gracias a los operadores ternarios, la sintaxis de un operador ternario es la siguiente:

condición ? acción condición verdadera : acción condición falsa;

El ejemplo de condicional que se ha mostrado anteriormente se expresaría de la siguiente forma:

var num = 20;
num % 2 == 0 ? console.log('Número par') : console.log('Número impar');
//'Número par'

A pesar de que los operadores ternarios son una forma sencilla y breve de escribir condicionales, no se recomienda utilizarlos cuando se aniden condiciones o las acciones a ejecutar sean muchas, puesto que el código se puede hacer ilegible.

Funciones

Las funciones son uno de los bloques de contrucción fundamentales de JavaScript.

Las funciones son un conjunto de instrucciones que realizan una tarea específica y se pueden reutilizar tantas veces como sea necesario; toman unos datos de entrada y devuelven una salida relacionada con la entrada.

La forma tradicional de definir una función es la declaración y su sintaxis es esta:

function nombre(parámetros de entrada) {
	declaraciones;
}

Otra forma de definir funciones son las expresiones de función, estas funciones se caracterizan por estar almacenadas dentro de una variable y se definen de la siguiente forma:

var variable = function (parámetros de entrada) {
	declaraciones;
}

Este segundo tipo de definición permite que la función se ejecute haciendo la llamada a la variable.

Continuando con el ejemplo mostrado anteriormente, a continuación definiremos una misma función utilizando ambos métodos:

//Declaración de función
function par(num) {
  if (num % 2 == 0) {
	  return 'Número par';
  } else {
	  return 'Número impar';
  }
}
console.log(par(10)); //'Número par'

//Expresión de función
var impar = function (num) {
  if (num % 2 == 0) {
	  return 'Número par';
  } else {
	  return 'Número impar';
  }
}
console.log(impar(5)); //'Número impar'

Dentro de las funciones de JavaScript, así como en las clases, se puede utilizar la palabra clave this. Esta palabra clave tiene un valor flexible y se utiliza para representar o llamar al objeto que una función está modificado.
